Projeto de Laboratório de Programação II

Esse repositório consiste no projeto desenvolvido para a disciplina de Laboratório de Programação II utilizando a linguagem de programação Java orientada a objetos.

Esse consiste em um sistema de cadastro de pesquisas acadêmicas, denominado "Psquiza", que tem como propósito organizar a evolução de uma pesquisa. No sistema é possível cadastrar Pesquisas, Alunos, Professores e Atividades e adicionar a eles outros artifícios que ajudam na organização do prosseguir do estudo.
